# fastapi-playground

## setup

```sh
pdm install
```

## dev

```sh
pdm run start
```

## debug

start and debug from `app/__main__.py`

![start and debug from `app/__main__.py`](./assets/debug_entry.png)

## docs

start the app then visit `http://localhost:8000/docs` or `http://localhost:8000/redoc`
